# 🧠 MindCare AI - Agentic Mental Health Ecosystem

MindCare AI is an Agentic AI-based mental health support ecosystem that uses specialized AI agents to understand user emotions, analyze thoughts, and provide supportive responses.

The project uses a local Large Language Model (Llama 3.2) through Ollama and provides an interactive Streamlit interface.

---

## ✨ Features

### 🧠 Mood Analysis Agent
- Detects user's emotional state
- Identifies possible emotions
- Provides supportive suggestions

### 📖 Journal Reflection Agent
- Analyzes journal entries
- Summarizes thoughts
- Provides reflection questions
- Suggests small positive actions

### 🤖 Local AI Processing
- Uses Llama 3.2 locally
- Runs without external API dependency
- Protects user privacy
